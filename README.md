# BrandMe.

## 포트폴리오 제작 플랫폼 | **BrandMe.**

개발자 직군의 포트폴리오 사이트를 만들어
개발자 자신만의 포트폴리오를 작성하고 관리하며,
다른 유저의 포트폴리오를 참고 및 상호 연락 할 수 있는 플랫폼입니다.
